# Bilder

Bilder werden in einen Ordner `media` abgelegt. Dieser Ordner befindet sich in der Regel auf der Ebene der entsprechenden md-Datei.

```
|-- <Directory>
      |-- <SubDirectory>
      |     |-- ...
      |-- media
      |     |-- bild.png
      |-- page.md
```

## Verweis auf ein Bild

```md
![Name des Bildes](media/bild.png)
```
