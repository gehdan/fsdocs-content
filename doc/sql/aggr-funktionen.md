# Aggregat-Funktionen

#### COUNT

`COUNT ( <Expression> )`

Liefert die Anzahl.

#### MIN

`MIN ( <Expression> )`

Liefert das Minimum.

#### MAX

`MAX ( <Expression> )`

Liefert das Maximum.

#### SUM

`SUM ( <Number Expression> )`

Liefert den Summe.

#### AVG

`AVG ( <Number Expression> )`

Liefert den Durchschnitt.

#### VAR

`VAR ( <Number Expression> )`

Liefert die Varianz.

#### STDEV

`STDEV ( <Number Expression> )`

Liefert die Standardabweichung.
