# Aggregat-Funktionen

## `COUNT ( <Expression> )`

Liefert die Anzahl.

## `MIN ( <Expression> )`

Liefert das Minimum.

## `MAX ( <Expression> )`

Liefert das Maximum.

## `SUM ( <Number Expression> )`

Liefert den Summe.

## `AVG ( <Number Expression> )`

Liefert den Durchschnitt.

## `VAR ( <Number Expression> )`

Liefert die Varianz.

## `STDEV ( <Number Expression> )`

Liefert die Standardabweichung.
